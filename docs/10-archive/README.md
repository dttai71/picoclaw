# 10-Archive: Centralized Legacy Archive

```yaml
directive: AI-NEVER-READ
reason: Contains outdated, superseded, or historical content
exception: User explicitly requests historical information
```

**Standard**: RFC-001 Legacy Document Organization (SDLC 6.0.5 MANDATORY)

All legacy content is organized by stage: `{NN}-Legacy/` where `{NN}` is the stage number.
Move deprecated documents to the appropriate `{NN}-Legacy/` folder instead of deleting them.
