## ADR Template

New ADRs should follow the format established in ADR-001:

1. Context — Why is this decision needed?
2. Decision Drivers — What constraints apply?
3. Considered Options — What alternatives were evaluated?
4. Decision — What was chosen and why?
5. Consequences — Positive, negative, risks, rollback strategy
